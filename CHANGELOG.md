## 版本说明

### 版本格式

版本号格式：`主版本号.次版本号.修订号`

- **主版本号**: 不兼容的 API 修改
- **次版本号**: 向下兼容的功能性新增
- **修订号**: 向下兼容的问题修正

### 计划的版本路线图

- **0.1.0** - Phase 1: 项目基础框架
- **0.2.0** - Phase 1: 拍卖阶段实现

- **0.4.0** - Phase 1: 航行阶段实现
